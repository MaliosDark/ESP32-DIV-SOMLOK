Kilaz Edition v4.7.0 — Web Installer

How to Flash

  1. Connect your ESP32-S3 (V2 board) to this computer via USB.
  2. Put the device in Boot/Flash mode: hold the BOOT button, press and release RESET, then release BOOT.
  3. Click the Install button below.
  4. Select the serial port for your ESP32-S3 when prompted.
  5. Wait for the flash to complete — the device will reboot automatically.
V2 boards only. This installer flashes the ESP32-S3 variant.

Requirements

  1. Browser: Google Chrome or Microsoft Edge (desktop). Web Serial is not supported in Firefox or Safari.
  2. USB cable: A data-capable USB cable (not charge-only) connected to the ESP32-S3.
  3. Drivers: No extra drivers needed. ESP32-S3 uses native USB.
